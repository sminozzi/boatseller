== Description ==
Boat Seller is a multilingual and multicurrency theme for dealership to manage, list and sell your new or used boats and pwc.
== Installation ==
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in boat seller in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
